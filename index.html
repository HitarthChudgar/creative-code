An attempt at creating a decode animating using css clip path property
The key trick here is to create 2 duplicate divs with same text; and adding a line height to space out text to bring it in the circle
Then finally adding a stoke to the text and adding keyframe animation stages using transform X and setting a linear animation with Xs duration we get this cool effect!
